Key Features

  • Private Connectivity: All traffic flows through your cloud provider's private network backbone
  • No Public Internet: Traffic never traverses the public internet, enhancing security
  • DNS Resolution: Private DNS ensures endpoints resolve to private IP addresses
  • Multi-Cloud Support: Works with AWS, Azure, and GCP Private Link services
  • Multiple Endpoints: Supports Kafka, Schema Registry, REST API, KSQL, and Control Center
  • Compliance: Helps meet regulatory requirements for data privacy and security
